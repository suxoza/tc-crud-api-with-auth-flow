# TS-Express CRUD Resource API With JSON Web Token(JWT)

* Server - Node Express
* Front - React / Redux
* Test - E2E / Cypress 

## Installation

# backend

```bash
cd server 
npm install
npm start
```

# frontend

```bash
cd frontend 
npm install
npm run dev 
```

## Run Test

* Note - While testing, the frontend should be up and running.

```bash
cd frontend  
npm run test
```
